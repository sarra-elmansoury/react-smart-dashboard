i# Smart E-Commerce Dashboard

A comprehensive React-based dashboard for managing products, users, and inventory status. This project demonstrates advanced React patterns including Context API, useReducer, Custom Hooks, and Performance Optimization.

## Features
- Add, Edit, and Delete products.
- Real-time search and stock filtering.
- Persistent data storage using LocalStorage.
- Optimized performance using `useMemo` and `useCallback`.
- Responsive UI built with React Bootstrap.
- Advanced Modal system using React Portals.


 
## Installation & Setup


 Install dependencies:
   npm install

 Run the project:
   npm start

## Technologies Used
- React.js
- React Bootstrap
- Context API
- LocalStorage API
